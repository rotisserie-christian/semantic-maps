### heuristics
- **`query_length`** - Age = tendency for longer vs shorter queries (younger = longer) [1,6].
- **`complexity`** - Age + domain expertise = how elaborate the initial query is [2,3,7].
- **`reformulation`** - Domain expertise + age = sticks to initial terms vs strategic reformulation / new keywords [2,4,5].
- **`specificity`** - Age + prior task knowledge = vague vs specific phrasing [2,6].
- **`patience`** - Age + cognitive flexibility = time per SERP, evaluation vs quick adjustment [3,7].
- **`word_choice`** - Age + expertise = vocabulary style (confident/casual vs formal, jargon vs lay terms) [6].
